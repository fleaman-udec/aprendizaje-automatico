Detección de Anomalías y Novedades¶

Introducción¶

Detección de anomalías¶

  • Los datos de entrenamiento contienen datos atpicos.
  • Los datos atpicos estn en regiones lejanas a los otros datos y de baja densidad.
  • Los modelos deben encontrar las regiones donde se ubican la mayora de los datos.

Detección de novedades¶

  • Los datos de entrenamiento no contienen datos atpicos.
  • Los modelos debern determinar si una nueva muestra es atpica o no.
  • Los datos atpicos podran eventualmente formar un grupo denso.

One-Class SVM¶

Adaptación Schölkopf¶

Proyecta datos a un espacio hiperdimensional y los separa del origen usando un hiperplano.

Un hiperparmetro $\nu$ fija la fraccin de datos atpicos en el entrenamiento.

Se maximiza la distancia entre hiperplano y origen, dejando datos por debajo como anomalas.

Adaptación Tax¶

Proyecta datos a un espacio hiperdimensional y los separa del origen usando una hiperesfera

Se puede usar un margen suave para admitir ciertas anomalas en los datos

Se minimiza el volumen de la esfera, dejando datos afuera como anomalas.

Bosque de aislamiento (Isolation Forest)¶

Generalidades¶

Utiliza aislamiento de los datos atpicos en vez de caracterizar los datos tpicos.

El aislamiento se realiza recursivamente mediante rboles binarios que eligen valores de corte para distintos atributos.

Formulación¶

  • Se define el camino hacia un dato como el nmero de particiones que un rbol debe realizar para poder aislarlo.
  • El algoritmo calcula el camino promedio hasta cada dato para varios rboles inicializados de forma aleatoria.
  • Se considera que los datos atpicos son los que tienen caminos ms cortos.
  • Un hiperparmetro "contamination" representa la proporcin de datos atpicos y se usa para estimar un umbral entre un "camino corto" y un "camino largo"

Factor de anomalía local (Local outlier factor, LOF)¶

Generalidades¶

Encuentra datos atpicos comparando densidades locales de los datos.

La densidad local se estima en base a los $k$ vecinos ms cercanos a un dato.

Comparte los conceptos de datos ncleo y datos alcanzables del mtodo DBSCAN.

Formulación¶

La densidad local de un dato se estima con el inverso de la suma de las distancias alcanzables de sus $k$ vecinos ms cercanos.

Se define el LOF de un dato como la razn entre la densidad local promedio de sus $k$ vecinos ms cercanos y su propia densidad local.

  • LOF>1 significa vecinos con densidad mayor al propio dato (dato atpico)
  • LOF<1 significa vecinos con densidad menor al propio dato (dato tpico)

fig.suptitle('Deteccin anomalas LOF', fontsize=16)

Sumario¶

  • Los mtodos de deteccin de anomalas y novedades permiten encontrar datos atpicos dentro de una muestra.
  • One-class SVM es una adaptacin del SVM para la aplicacin en deteccin de anomalas.
  • El bosque de aislamiento permite utilizar rboles de decisin para aislar anomalas o novedades.
  • El factor de anomala local es una adaptacin de DBSCAN para detectar anomalas o novedades.
